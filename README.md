# 🦁 ZooNak - Biodiversidade do Brasil

Este projeto é uma aplicação front-end interativa focada na conscientização sobre a biodiversidade brasileira. Desenvolvido como parte do aprendizado em tecnologias web, o projeto foca em experiência do usuário e interatividade.

### ⚙️ Funcionalidades Principais
* **Navegação por Abas:** Organização de conteúdo de forma limpa e intuitiva.
* **Quiz Dinâmico:** Teste de conhecimentos com sistema de pontuação em tempo real via JavaScript.
* **Agendamento:** Formulário com calendário integrado e validações de campos.
* **Design Responsivo:** Adaptado para diferentes tamanhos de tela.

### 🛠 Tecnologias e Bibliotecas
* **HTML5 / CSS3**
* **JavaScript (ES6+)**
* **Slick Slider:** Para carrosséis de imagens e banners.
